To run:
(In terminal)
javac Server.java

javac Game.java

javac ClientGUI.java

java Server <port number> <hostname>

java ClientGUI

(In ClientGUI)
Once ClientGUI is opened, enter the name of the game you want to track the price of.
GUI will then prompt you to enter the hostname and port number to connect to the server.
Once all requirements have been met, the GUI will display all pricing data about the specified game.
